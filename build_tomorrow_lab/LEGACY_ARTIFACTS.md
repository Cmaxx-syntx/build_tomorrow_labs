# Legacy artifacts with old branding

These files in `public/` contain the old "Build Tomorrow Lab" branding and are **not used** by the current Next.js app. The app now serves from `app/` with the new "Builder Support Fund" branding.

- `public/index.html` — old static homepage with "Build Tomorrow Lab" title
- `public/support.js` — old support script

These are safe to delete. They are left here for now in case you want to keep them as a backup. To remove them:

```bash
rm public/index.html public/support.js
```
